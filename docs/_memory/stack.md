# Tech Stack

- **Linguagem:** Java 17
- **Framework Core:** Spring Boot 3.3.0
- **Camada de Dados:** Spring Data JPA / Hibernate
- **Banco de Dados:** MySQL 8.0
- **Segurança:** Spring Security + JWT (Stateless)
- **Infraestrutura:** Docker e Docker Compose
- **Design Pattern:** Layered Architecture (Controllers, Services, Repositories, Models, DTOs)
